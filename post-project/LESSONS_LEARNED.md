# Lessons Learned — Wnioski z Projektu

> Konkretna lista wniosków do zastosowania w przyszłych projektach.
> W odróżnieniu od retrospektywy — to plik "do zabrania" do kolejnego projektu.
>
> **Jak wypełniać:** po zakończeniu projektu (lub po każdym etapie) zapisuj obserwacje
> w formacie poniżej. Jeśli wniosek powoduje zmianę w szablonie — zaktualizuj też
> `draft/project-template/` i zaznacz co zostało wdrożone.

## Format wpisu

```
### [Temat]
**Obserwacja:** [Co zaobserwowaliśmy — konkretne zdarzenie, nie ogólnik]
**Wniosek:** [Co z tego wynika dla przyszłych projektów]
**Akcja na przyszłość:** [Konkretna zmiana w procesie / template — lub "brak, wystarczy świadomość"]
**Wdrożone w szablonie:** [ ] tak / [ ] nie / [ ] nie dotyczy
```
