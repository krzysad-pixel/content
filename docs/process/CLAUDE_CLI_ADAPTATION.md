# CLAUDE_CLI_ADAPTATION

## Cel
Ten dokument wskazuje, co może wymagać dopasowania do lokalnej wersji Claude CLI.

## Elementy do dopasowania
1. Dokładna komenda uruchamiająca prompt.
2. Ewentualny sposób przekazywania pliku promptu.
3. Obsługa sesji / logów CLI.
4. Format hooków, jeśli lokalna wersja CLI różni się od przyjętego szkicu.

## Zasada
Nie zmieniać modelu dokumentacji i pipeline'u tylko dlatego, że lokalna składnia CLI różni się w jednym poleceniu. Dopasować cienką warstwę uruchomieniową.
